División del trabajo

El contenido del curso fue extenso, por lo que se dividió entre integrantes del equipo para una mejor organización. Cada semana fue asignada a un responsable distinto para asegurar profundidad y claridad en el contenido presentado.

Estilo del sitio

Si bien este sitio no incluye aún estilos visuales, se recomienda aplicar un CSS atractivo para mejorar la presentación y usabilidad. Se pueden usar fuentes legibles, colores suaves, espaciado adecuado y navegación clara.
