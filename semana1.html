Desarrollo Web vs. Diseño Web

Es importante diferenciar entre desarrollo web y diseño web, aunque a menudo se confunden.

  • Diseño Web: Se enfoca en la apariencia y la experiencia del usuario de un sitio web. Incluye el diseño, la navegación, la paleta de colores, y el diseño gráfico del logo, priorizando la estética y la facilidad de uso.
  • Desarrollo Web: Se encarga de la funcionalidad y las características del sitio. Implica la programación del "backend", como los sistemas de registro, la gestión de contenido y las aplicaciones de bases de datos.

Arquitectura de Software: Cliente-Servidor

La arquitectura de software define la estructura, el funcionamiento y la interacción de las partes de un sistema. En el desarrollo web, la arquitectura fundamental es la Cliente-Servidor.

Esta arquitectura consiste en un cliente (como un navegador web) que solicita información o servicios a un servidor, el cual procesa la petición y envía una respuesta. Por ejemplo, al visitar Wikipedia, tu navegador es el cliente que pide un artículo al servidor de Wikipedia.

Componentes:

  • Cliente: Es un programa que inicia conexiones para solicitar servicios y tiene una vida útil finita.
  • Servidor: Es un programa que ofrece un servicio en una red, esperando peticiones de los clientes y con una vida útil "interminable".

Roles y Etapas en el Desarrollo Web

El desarrollo web moderno implica un trabajo en equipo con diferentes roles especializados, a diferencia del antiguo rol único de "Webmaster". Algunos roles clave son:

  • Desarrollador/a Frontend: Programa la lógica del lado del cliente, se encarga de la interfaz y consume datos del servidor.
  • Desarrollador/a Backend: Programa la lógica del lado del servidor, conectándose con la base de datos para responder a las solicitudes.
  • Diseñador/a UI/UX: Se ocupa del aspecto visual y la experiencia del usuario, utilizando software de diseño.
  • Desarrollador/a FullStack: Tiene habilidades tanto en frontend como en backend.

Etapas del Desarrollo:

  1. Planificación: Se documentan los requisitos y tecnologías.
  2. Diseño: Se define la interfaz gráfica y la experiencia de usuario.
  3. Desarrollo: Se construye y programa la aplicación.
  4. Despliegue: La aplicación se pone a disposición de los usuarios.
  5. Soporte: Se realizan correcciones y mantenimiento.

Tecnologías Fundamentales

¿Qué son HTML y CSS?

  • HTML (HyperText Markup Language): Es el lenguaje de marcado para crear y estructurar el contenido de una página web. Define elementos como textos, imágenes y enlaces.
  • CSS (Cascading Style Sheets): Es el lenguaje usado para describir la presentación y el estilo visual de los documentos HTML.

¿Cómo incluir JavaScript?

Para añadir interactividad, se utiliza JavaScript. Se crea un archivo con extensión .js y se incluye en el HTML mediante la etiqueta <script>.

Herramientas del Desarrollador

Las herramientas esenciales para el desarrollo web incluyen un editor de texto y las herramientas de inspección del navegador.

  • Editor de Texto: Un programa para escribir código en texto plano, como Visual Studio Code.
  • Herramienta del Desarrollador del Navegador: Permite inspeccionar el código HTML, CSS y JavaScript de una página, depurar errores y probar cambios en tiempo real. Se puede acceder generalmente haciendo clic derecho y seleccionando "Inspeccionar".
