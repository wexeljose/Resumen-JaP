1. Inclusión de JavaScript

El código JavaScript se puede incluir directamente en el HTML con la etiqueta <script> o mediante un archivo externo usando src: 

<script src="script.js"></script>

Se recomienda ubicar los scripts al final del <body> o usar defer para mejorar el rendimiento.

💡 Buena práctica: Usar defer permite que el script se cargue en paralelo con el HTML pero se ejecute después de que el DOM esté completamente construido.

2. Estructura de Código

El código se compone de sentencias, separadas con punto y coma. Se recomienda una sentencia por línea.

Existen dos tipos de comentarios:

  • De una línea: // comentario
  • Multilínea: /* comentario */

3. Funciones y Variables

Las funciones permiten reutilizar código. Se definen con la palabra function: 

function saludar() {
  alert("Hola");
}

Variables: let y const son las formas modernas de definir variables. let permite reasignar valores; const no.

4. DOM (Document Object Model)

El DOM es una representación en forma de árbol de los elementos HTML.

Tipos de nodos principales:

  • Document: raíz del árbol
  • Element: etiquetas HTML
  • Text: contenido textual
  • Attr: atributos de las etiquetas

Acceso a elementos

  • getElementById('id')
  • getElementsByClassName('clase')
  • getElementsByTagName('etiqueta')
  • getElementsByName('name')

Manipulación

  • Crear elementos: document.createElement()
  • Añadir texto: document.createTextNode()
  • Añadir al DOM: appendChild()
  • Eliminar: element.remove()

Demo: Manipulación del DOM

Clases

Usamos classList para manejar clases:

  • add('clase')
  • remove('clase')
  • toggle('clase')
  • contains('clase')

5. JSON (JavaScript Object Notation)

Formato de texto ligero para intercambio de datos. Es similar a los objetos en JavaScript. 

{
  "nombre": "Ana",
  "edad": 25,
  "habilidades": ["HTML", "CSS", "JS"]
}

Métodos útiles:

  • JSON.parse(): convierte un texto JSON a objeto JS
  • JSON.stringify(): convierte un objeto JS a texto JSON

6. HTTP y Fetch

JavaScript puede comunicarse con servidores mediante HTTP. Los métodos más comunes:

  • GET: obtener información
  • POST: enviar datos

Fetch

fetch() es la forma moderna de hacer solicitudes HTTP. Devuelve una promesa. 

fetch('https://api.example.com/datos')
  .then(response => response.json())
  .then(data => console.log(data));

Ejemplo con async/await

async function cargarDatos() {
  const respuesta = await fetch('datos.json');
  const datos = await respuesta.json();
  console.log(datos);
}
