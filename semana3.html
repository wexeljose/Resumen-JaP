Resumen de CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) permite separar el contenido del diseño en una página web. Se puede aplicar directamente en el HTML o mediante archivos externos.

Formas de aplicar estilos

  • En línea: usando el atributo style dentro de una etiqueta HTML.
  • Interno: mediante la etiqueta <style> en el HTML.
  • Externo: enlazando un archivo .css con <link>.

Sintaxis

Una regla CSS está formada por un selector y un bloque de declaración. Cada declaración tiene una propiedad y un valor, separados por dos puntos y finalizados con punto y coma.

Tipos de selectores

  • E: todos los elementos E.
  • *: todos los elementos.
  • E.clase: elementos E con clase específica.
  • E#id: elemento E con ID específico.
  • E F: F dentro de E (descendiente).
  • E > F: F hijo directo de E.
  • E + F: F inmediatamente después de E.
  • E, F: aplica a ambos.

Estilos en cascada

El navegador aplica los estilos según una jerarquía:

  1. Estilos en línea.
  2. Estilos en <style>.
  3. Hojas de estilo externas.
  4. Estilos por defecto del navegador.

Si hay conflictos, gana el más específico o el último definido.

Tamaños y distancias

Se definen usando unidades como px, em, %, vh, vw, entre otras.

El tamaño puede expresarse en valores absolutos o relativos al contenedor o al tipo de letra.
